# UC-05: EXO 詳細設定

## 概要

AviUtl 向け EXO ファイルの出力設定（フォント・サイズ・配置・スタイル等）の変更を確認する。

---

## ユースケース一覧

### UC-05-01: デフォルト値の確認

**前提条件**: アプリが起動し pywebview API が準備完了している

**期待結果**:
- 「フォントサイズ」が `34`
- 「字間」が `0`
- 「行間」が `0`
- 「表示速度」が `0`
- 「寄せ方向」が「中央」 (value=4)
- 「X位置」が `0`
- 「Y位置」が `0`
- 「1行の最大文字数」が `20`
- 「縁取り」チェックボックスが ON
- 「太字」「斜体」チェックボックスが OFF

### UC-05-02: フォント選択

**前提条件**: フォント一覧が読み込まれている

**操作**: 「フォント」セレクトで別のフォント名を選択する

**期待結果**:
- 選択したフォント名が `#exo-font` の値に反映される

### UC-05-03: フォントサイズの変更

**前提条件**: アプリが起動している

**操作**: 「フォントサイズ」入力を `48` に変更する

**期待結果**:
- `#exo-font-size` の値が `48` になる

### UC-05-04: 寄せ方向の変更

**前提条件**: アプリが起動している

**操作**: 「寄せ方向」セレクトで「左上」(value=0) を選択する

**期待結果**:
- `#exo-align` の値が `"0"` になる

### UC-05-05: 太字・斜体・縁取りの切り替え

**前提条件**: アプリが起動している

**操作A**: 「太字」チェックボックスをクリックする  
**操作B**: 「斜体」チェックボックスをクリックする  
**操作C**: 「縁取り」チェックボックスをクリックする（OFF → ON または ON → OFF）

**期待結果**:
- 各チェックボックスの状態が切り替わる

### UC-05-06: 設定変更によるプレビューへの反映

**前提条件**: モックデータが注入済みでプレビューが表示されている

**操作**: フォントサイズを変更し「✔ 適用」を押す（またはプレビューが自動更新される）

**期待結果**:
- プレビュー画像が更新される（`#preview-image` の `src` 属性が変わる）

### UC-05-07: 最大文字数による折り返し

**前提条件**: セグメントが選択され、長いテキストが設定されている

**操作**: 「1行の最大文字数」を `10` に変更する

**期待結果**:
- プレビューで字幕が折り返して表示される

## テスト実装メモ

```python
def test_exo_defaults(page):
    assert page.input_value("#exo-font-size") == "34"
    assert page.input_value("#exo-spacing-x") == "0"
    assert page.locator("#exo-soft-edge").is_checked()
    assert not page.locator("#exo-bold").is_checked()

def test_exo_font_size_change(page):
    page.fill("#exo-font-size", "48")
    page.dispatch_event("#exo-font-size", "change")
    assert page.input_value("#exo-font-size") == "48"
```
